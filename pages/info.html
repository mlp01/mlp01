macOS dock

Sujet

Reproduction du dock de macos pour windows 10 & 11.

Contexte

Projet personnel démarré en 2019, arrêté en 2021.

Fonctionnalités

  • Lancer, minimiser et fermer des applications
  • Indicateur d’applications ouvertes
  • Menu déroulant avec options pour chaque application
  • Paramétrage de chaque application dans le dock : changer l’icône et sa taille, le nom de l’application et le chemin d’exécution
  • Effet de flou dynamique selon le contenu se trouvant derrière le dock
  • Fenêtre de paramètres permettant de définir l’opacité, la taille, la magnification et le thème (clair ou sombre) du dock
  • Fenêtre de gestion de la luminosité des écrans

Implémentation

Languages de programmation : principalement C# et XAML, quelques éléments en C++
IDE : Visual Studio 2019 et Visual Studio Code
